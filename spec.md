# LMS Auto-Watch & Transcript — Spec

> 작성: 2026-03-02 20:42
> 갱신: 2026-03-03
> 상태: 초안 — 리뷰 필요

## 한 줄 목적

숭실대 LMS 사전녹화 강의를 자동 재생(출석 처리)하고, 재생 중 스크립트를 추출해두는 도구. 두 가지 모드(자동 수강 / 다운로드)를 제공.

## 왜 필요한가

- 시간이 없고 귀찮음 — 사전녹화 강의를 직접 앉아서 볼 여유가 없음
- 깜빡 놓치면 성적 깎임 — 마감 전에 출석 처리가 안 되면 감점
- 스크립트가 있으면 나중에 빠르게 복습 가능

## 사용자

나 (숭실대 학부생, 1명). 개인 도구.

## 사용 모드

### 모드 1 — 자동 수강 (watch)

미수강 동영상만 자동 재생 + 다운로드/전사. 수강완료 과목은 스캔하지 않음.

```
$ python -m src.auto_watch

모드를 선택하세요:
  [1] 자동 수강 — 미수강 동영상 재생 + 다운로드/전사
  [2] 다운로드  — 과목 선택 → 강의 다운로드/전사만

번호 (1/2): 1
```

**흐름:**
1. 실행 → 모드 선택 → SSO 자동 로그인
2. 마이페이지에서 `videoCount > 0`인 과목만 주차학습 페이지 진입
3. movie 타입 강의만 수집 (PDF/과제 제외)
4. 사용자가 번호 선택 (복수 선택, `all`, `q`로 종료)
5. 선택한 강의를 순차 재생 + 병렬로 다운로드/전사
6. 결과 요약 출력

### 모드 2 — 다운로드 (download)

과목을 먼저 선택 → 해당 과목만 lazy 스캔 → 다운로드/전사만 수행.

```
번호 (1/2): 2

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  과목 8개:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [1] 자료구조 (수강 완료)
  [2] 감성지능리더십 (미수강 2개)
  ...

번호 / all / q: 1
```

**흐름:**
1. 실행 → 모드 선택 → SSO 자동 로그인
2. 마이페이지에서 과목 목록 표시 (주차학습 페이지 미진입 — 빠름)
3. 사용자가 과목 선택 → 해당 과목만 주차학습 페이지 진입
4. movie 타입 강의 중 선택 → 다운로드/전사만 수행 (재생/출석 없음)

### 모드 B — 자동 (cron, 미구현)

OpenClaw + cron으로 주기적 실행. 사용자 개입 없이 전체 자동.

```
$ python -m src.auto_watch --auto
```

**흐름:**
1. cron 트리거 (예: 매주 월요일 새벽)
2. 로그인 → 시작일 지난 미수강 강의 전부 재생
3. 스크립트 추출
4. 텔레그램으로 결과 알림 ("3개 수강 완료, 스크립트 3개 저장")

## 파이프라인

강의 1개당 두 가지 작업이 동시에 진행:

```
강의 재생 시작
  ├─ [출석] 1x 배속 재생 → 완료 이벤트 대기 → 출석 인정
  └─ [스크립트] 영상 다운로드 → faster-whisper로 음성→텍스트 추출
```

- **출석**: 브라우저에서 1x 재생. 서버가 누적 시간 추적하므로 끝까지 재생 필수
- **스크립트 추출**: 영상 파일 다운로드 → faster-whisper (로컬, API 불필요)
- **요약 (optional)**: Gemini 무료 tier 또는 수동으로 Claude에 붙여넣기. 핵심 스코프 아님

### 저장 경로

```
output/
  └─ {과목명}/
      ├─ {강의명}.mp4    # 영상 파일
      └─ {강의명}.txt    # 스크립트 (음성→텍스트)
```

예: `output/자료구조/chap1_1.mp4`, `output/자료구조/chap1_1.txt`

## 성공 기준

- [x] watch 모드: 선택한 강의만 재생되고, 미선택 강의는 건드리지 않음
- [x] watch 모드: 재생 완료 후 LMS에서 출석 인정 확인
- [x] watch 모드: 스크립트 텍스트 파일과 영상 파일이 output/ 에 저장됨
- [x] watch 모드: 수강완료 과목은 주차학습 페이지 스킵
- [x] download 모드: 과목 선택 → lazy 스캔 → 다운로드/전사만
- [x] movie 타입 필터: PDF/과제 항목 제외
- [ ] 모드 B: `--auto` 플래그로 전체 자동 실행
- [ ] 모드 B: 텔레그램 알림 수신
- [x] 시작일 전 강의는 재생 대상에서 제외
- [x] 1x 배속 유지 (서버 배속 감지 통과)

## 제약

- **1x 배속 필수**: LMS 서버가 `invalidPbrate`로 배속 감지. 초과 시 부정행위 처리
- **동시 수강 불가**: 창 여러 개로 동시 재생하면 안 됨. 순차 재생만
- **headed 브라우저**: 헤드리스면 LTI 플레이어가 완료 이벤트 안 보낼 수 있음
- **SSO 로그인**: `page.type` + JS `btn_click` 방식 (Playwright `fill`이 안 먹힘)

## 범위 밖 (Out of Scope)

- LLM 요약 자동화 (스크립트 추출까지만. 요약은 수동)
- 과제/퀴즈 자동 제출
- 실시간 강의 (Zoom 등) 자동 참석
- 모바일 앱

## 기술 스택

- **언어**: Python 3.11+
- **브라우저 자동화**: Playwright (headed, Chromium)
- **음성→텍스트**: faster-whisper (CTranslate2, 로컬)
- **자동화 트리거**: OpenClaw (cron + 텔레그램)
- **기존 코드**: lms-summarizer 레포 (Playwright, faster-whisper 이미 셋업됨)

## 마일스톤

1. **M1 — 수동 선택 재생**: CLI 목록 → 번호 선택 → 재생 → 출석 확인 ✅
2. **M2 — 스크립트 추출 연결**: 재생 중 영상 다운로드 + faster-whisper 파이프라인 연결 ✅
3. **M2.5 — 수강완료 강의 다운로드**: 아래 상세 스펙 참고 ✅
4. **M3 — watch/download 모드 분리**: 모드 선택 UI + movie 필터 + lazy 스캔 ✅
5. **M4 — 자동 모드**: `--auto` 플래그 + 전체 자동 재생
6. **M5 — OpenClaw + 텔레그램**: cron 트리거 + 결과 알림

# M2.5 — 수강완료 강의 다운로드 지원

## 문제

현재 도구는 **미수강 강의만** 대상으로 동작한다.

| | 자동 수강 처리 | 다운로드 + 전사 |
|---|---|---|
| 미수강 강의 | O | O |
| 수강완료 강의 | 불필요 | **X ← 문제** |

이미 들은 강의의 영상/스크립트를 확보할 수 없다. 두 가지 필터가 원인:

1. **과목 필터** (`courses.py:46`): `videoCount > 0`인 과목만 → 전부 수강 완료된 과목은 안 뜸
2. **강의 필터** (`courses.py:132`): `isCompleted === true`인 강의 제외

## 해결 방향

**모든 강의를 보여주되, 상태에 따라 동작을 분기한다.**

- **미수강** → 기존대로: 자동 재생 + 완료 대기 + 다운로드 + 전사
- **수강완료** → 재생 버튼 클릭(URL 캡처용) → 즉시 정지 → 다운로드 + 전사만

---

### 1. 과목 조회

**함수명**: `get_unwatched_courses()` → `get_courses()`

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 필터 | `videoCount > 0`만 반환 | 모든 과목 반환 (필터 제거) |
| `videoCount` | 필터링 용도 | UI 표시용으로만 유지 |
| 빈 결과 | "모든 강의 수강 완료" | "수강 중인 과목이 없습니다" |

**로그 출력**:

```

  • 컴퓨터네트워크 — 미수강 5개
  • 데이터베이스 — 미수강 3개
  • 운영체제 — 수강 완료
  • 알고리즘 — 수강 완료
  • 소프트웨어공학 — 미수강 1개
```

### 4. 플레이어 분기

**함수명**: `watch_lecture()` → `process_lecture()`

`isCompleted` 여부에 따라 분기:

```
┌─────────────────────────────────────────────┐
│              공통                             │
│  페이지 이동 → iframe 대기 → 이어보기 처리    │
│  → 재생 버튼 클릭 → 비디오 URL 캡처           │
└──────────────┬──────────────────┬────────────┘
               │                  │
         미수강 강의          수강완료 강의
               │                  │
      진행 모니터링 루프      즉시 비디오 정지
      (완료 감지까지 대기)         │
               │                  │
┌──────────────┴──────────────────┴────────────┐
│              공통                             │
│  다운로드 + 전사 → 결과 반환                   │
└─────────────────────────────────────────────┘
```

**수강완료 강의 처리 흐름**:

1. 강의 페이지 진입
2. iframe 대기 + 이어보기 다이얼로그 → "아니오"
3. **재생 버튼 클릭** (비디오 URL 캡처 트리거)
4. URL 캡처 대기 (최대 5초)
5. **즉시 `video.pause()`** — 불필요한 재생 방지
6. 다운로드 + 전사 시작
7. 진행 모니터링 **스킵**
8. 결과 반환

**로그**:

```
미수강:   [PLAY] 2주차 정렬 (35:12)
수강완료: [DL] 3주차 탐색 (수강완료 — 다운로드만)
```
